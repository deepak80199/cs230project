# cs230project
This repo is for CS230 project
```
 |- report : Final Report for the project
 |- final : Code and experimentation
        |- lstm : contains the scripts to train the LSTM model (with hyperparameter serach)
        |- model : contains saved models
        |- output : contains saved outputs

        |- utils : contains utilities require to run project
        |- yoga_data_large_labeled : Data used for training
        |- notebooks : Notebooks for experimentation and running the project 
        |- requirement.txt : file containing the pip libraries (python=3.9)
```
